Project Overview

This project focuses on analyzing retail sales data to discover business insights, clean raw datasets, and visualize trends that help guide business decisions.
It demonstrates practical data analysis using Python, Pandas, and Power BI.

 Objectives

Clean and prepare raw retail sales data

Perform exploratory data analysis (EDA)

Identify top-selling products and cities

Calculate revenue performance

Visualize trends using Power BI dashboards

Publish and version control the project on GitHub

 Tools & Technologies

Pandas

NumPy

Matplotlib

Power BI

Git & GitHub
